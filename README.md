# 📊 Painel GLPI (Dashboard Plugin)

> **Um projeto da comunidade, para a comunidade.**

Este repositório reúne a versão atualizada e aprimorada do plugin original **Dashboard**, com o intuito de dar continuidade a esta ferramenta essencial para administradores e analistas de TI.

## 🤝 História e Créditos

Este projeto nasceu do excelente trabalho de mantenedores anteriores que ajudaram a construir a comunidade GLPI:
1. Criado originalmente por **Stevenes Donato**.
2. Posteriormente bifurcado (fork) e mantido pela **Service TIC**.

Como o desenvolvimento por parte dos mantenedores anteriores foi paralisado, eu, **SnakeBigBoss**, decidi assumir a frente do projeto. O objetivo é garantir que essa incrível contribuição não morra, mantendo o legado dos amigos que iniciaram essa jornada e trazendo o plugin para as necessidades atuais.

## ✨ O que há de novo nesta versão?

Além de preservar a estabilidade do projeto original, esta nova etapa traz uma série de atualizações significativas:

- 🛠️ **Novas Opções de Configuração:** Mais flexibilidade para a administração do seu painel e ajustes de ambiente.
- 🧹 **Limpeza e Otimização do GLPI:** Rotinas aprimoradas para manutenção, mantendo o seu sistema mais leve e responsivo.
- 📈 **Novos Gráficos:** Métricas e visuais atualizados, facilitando a extração de relatórios e a análise de dados do setor.
- 💻 **Tela de Chamados Totalmente Remodelada:** Uma interface modernizada, com design focado em entregar a melhor usabilidade e clareza para o acompanhamento dos tickets.

## 🚀 Contribuições e Feedback

A comunidade é o que mantém este projeto vivo! Sinta-se à vontade para testar em seu ambiente, e utilize a aba **Issues** do GitHub para relatar bugs, tirar dúvidas ou sugerir novas ideias.
